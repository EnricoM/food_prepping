# Quick Translation Helper

## Fastest Options:

### Option 1: Google Translate (Free, Quick)
1. Open: https://translate.google.com
2. Upload `lib/i18n/en.i18n.json`
3. Select target language
4. Download translation
5. Save as `{lang}.i18n.json`

### Option 2: DeepL (Better Quality, Free)
1. Open: https://www.deepl.com/translator
2. Copy content from `en.i18n.json`
3. Paste and translate
4. Copy result to new file

### Option 3: Translation Platform (Best for Teams)
- **Crowdin**: Free for open source - upload once, get community translations
- **Lokalise**: Professional platform with quality checks

### Option 4: API Script (Automated)
Create a script using:
- Google Cloud Translation API
- DeepL API
- Microsoft Translator API

See `TRANSLATION_GUIDE.md` for detailed instructions.
